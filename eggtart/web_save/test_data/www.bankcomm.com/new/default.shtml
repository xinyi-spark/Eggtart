境外分支机构

截止2015年6月末，交通银行共在境外13个国家和地区设立了15家境外分支机构，包括香港、纽约、旧金山、东京、新加坡、首尔、法兰克福、悉尼、布里斯班、澳门、胡志明市和台北分行，交通银行（英国）有限公司及交通银行（卢森堡）有限公司，多伦多代表处，境外经营网点合计56个，形成了“一体两翼”的境外机构布局，境外银行机构资产总额达人民币8,115.80亿元，较年初增长31.42%，实现净利润人民币20.66亿元。与全球144个国家和地区的1,639家同业建立代理行关系。

交通银行澳门分行成立于2007年11月，持有商业银行全牌照，致力于以“财富管理”为特色，为澳门工商界及个人客户提供优质的银行服务，主要有: 公司客户各类存款、贷款、国际结算、贸易融资、资产管理、托管、代客外汇买卖、衍生工具买卖等；个人客户各类存款、贷款、楼宇按揭、外汇兑换等；同时还提供证券、基金、保险、私人银行、信托、等投资理财服务。

交通银行香港分行成立于1934年，在港持续经营超过80年，持有商业银行全牌照下辖43家网点。交通银行香港分行为客户提供各类银行服务，主要有: 公司客户各类存款、贷款、国际结算、贸易融资、外汇兑换、衍生工具买卖等；个人客户楼宇按揭、私人贷款、信用卡、保管箱、外汇兑换、衍生工具买卖等；并提供证券、保险、私人银行和信托等相关的金融服务。特色金融服务有：跨境银行服务、IPO综合银行服务和跨境人民币服务等。

交通银行台北分行成立于2012年7月，持有批发银行牌照。交通银行台北分行主要为两岸公司客户提供各类存款、贷款、国际结算、贸易融资、外汇兑换、跨境人民币等银行服务。

交通银行纽约分行成立于 1991年11月，下辖旧金山分行，业务覆盖美国东西海岸。交通银行纽约分行持有批发银行牌照，为客户提供各类银行服务，主要有: 公司客户各类存款、贷款、国际结算、贸易融资、外汇兑换、美元清算及跨境人民币等。

交通银行旧金山分行成立于2011年11月，是纽约分行的下属分行，持有批发银行牌照，为客户提供各类银行服务，主要有: 公司客户各类存款、贷款、国际结算、贸易融资、外汇兑换、美元清算及跨境人民币等。

交通银行多伦多代表处成立于2014年，标志着交通银行在北美地区的机构布局进一步完善，将为中加两国的公司客户提供业务咨询及联络。

交通银行首尔分行成立于2005年8月，持有商业银行全牌照，2013年获任韩国地区跨境人民币清算行。主要为公司客户提供存款、贷款、汇款、贸易结算、贸易融资、代客外汇买卖等服务；为个人客户提供存款、汇款等服务；为同业客户提供人民币及外币的清算及汇款服务。特色业务有：韩元兑换、韩元汇款服务等。

交通银行东京分行成立于1995年12月，持有商业银行全牌照，为客户提供各类银行服务，主要有: 公司客户各类存款、贷款、国际结算、贸易融资、外汇兑换、衍生工具买卖、日元清算及跨境人民币等；个人客户各类存款、贷款、汇款、个人按揭贷款等银行服务。特色金融服务有：个人按揭贷款。

交通银行新加坡分行成立于1996年9月，持有批发银行牌照，主要为公司客户提供各类存款、贷款、汇款、贸易融资、代客外汇买卖、跨境人民币等银行服务。

交通银行胡志明市分行成立于2011年2月，持有商业银行全牌照。分行为中越企业提供全面金融服务，为赴越南投资、承包工程以及开展中越经贸交流的中资客户提供全方位的支持，主要有：公司客户各类存款、贷款、国际结算、贸易融资业务，个人客户存款、汇款、外汇兑换业务等。

交通银行（英国）股份有限公司（简称“英国子行”）成立于2011年11月，位于伦敦，持有批发银行牌照。英国子行为公司客户提供各类存款、贷款、国际结算、贸易融资、外汇兑换、衍生工具买卖、跨境人民币等银行服务。

交通银行法兰克福分行成立于2007年10月，持有商业银行全牌照，为客户提供各类银行服务，主要有: 公司客户各类存款、贷款、国际结算、贸易融资、外汇兑换、代客外汇买卖、担保及咨询、跨境人民币等。

交通银行（卢森堡）有限公司成立于2015年5月，持有商业银行全牌照。卢森堡子行可为公司客户提供各类存款、贷款、国际结算、贸易融资、外汇兑换、跨境人民币等银行服务。

交通银行悉尼分行成立于2011年11月，下辖布里斯班分行。交通银行悉尼分行持有批发银行牌照，，为公司客户提供各类存款、贷款、国际结算、贸易融资、外汇兑换、跨境人民币等银行服务。

交通银行布里斯班分行成立于2015年4月，持有批发银行牌照，为悉尼分行下属分行，为公司客户提供各类存款、贷款、国际结算、贸易融资、外汇兑换、跨境人民币等银行服务。
